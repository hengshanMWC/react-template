# react-template
使用`react@v18、vite@v4、antd@v4、unocss、zustand@v4`搭建的项目模板

提供简单的后台风格布局，抽离了 axios 拦截
## 构建 Start

```javascript
pnpm install       // 安装依赖模块
pnpm dev           // 运行开发环境
pnpm build         // 正式打包，生成最终代码
pnpm preview       // 本地运行正式打包后的最终代码
pnpm format        // 一键格式化代码
pnpm commit        // 代替 git commit
```
